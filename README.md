Source code for my personal academic website at [leeannvc.com](https://leeannvc.com/).

## About

This repository builds and deploys my personal website, where I share my research, publications, and CV. I am a variationist sociolinguist with expertise in linguistic research, dataset curation, and applied data science.

## Built With

This site is a customized version of the [Academic CV template](https://github.com/HugoBlox/hugo-theme-academic-cv) from [Hugo Blox](https://hugoblox.com/), created by [George Cushen](https://georgecushen.com). It is built and edited in VS Code and deployed via [Netlify](https://www.netlify.com/).
